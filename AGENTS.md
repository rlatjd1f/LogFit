# AGENTS.md

## 프로젝트 메모

LogFit은 HTML, CSS, Vanilla JavaScript로 구성된 초경량 정적 PWA입니다.

## 커밋 컨벤션

본 프로젝트의 커밋 메시지는 아래 prefix를 사용합니다.

- `feat`: 새로운 기능 추가
- `fix`: 버그 수정 및 시스템 안정화
- `docs`: 문서 수정 (README, GEMINI 등)
- `style`: 디자인 및 레이아웃 조정
- `refactor`: 코드 구조 개선

커밋 내용은 한국어로 작성합니다. prefix는 영어 규칙을 유지하고, prefix 뒤 설명은 간결한 한국어 문장으로 작성합니다.

예시:

- `feat: 루틴 생성 기능 추가`
- `fix: 휴식 타이머 종료 오류 수정`
- `docs: README 실행 방법 보강`

하나의 커밋에는 하나의 논리적 변경만 포함합니다.

## 원격 푸시 규칙

- 사용자가 커밋을 요청할 때 또는 에이전트가 개발 및 오류 수정을 완료하여 로컬 커밋을 생성하는 모든 Git 작업 시, 로컬 커밋 완료 즉시 원격 저장소로의 푸시(`git push`)까지 누락 없이 함께 실행하여 원격 저장소를 실시간 최신 상태로 항상 동기화합니다.
