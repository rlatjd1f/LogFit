LogFit 개인정보처리방침

시행일: 2026년 7월 21일

LogFit은 사용자의 운동 기록을 기기 안에서 처리하는 오프라인 운동 일지입니다. 개발자는 앱을 통해 개인정보나 운동 기록을 수집, 전송, 판매 또는 제3자와 공유하지 않습니다.

처리하는 데이터

사용자가 입력한 루틴, 운동명, 중량, 반복 수, 세트 기록, 앱 설정 및 타이머 상태는 기기의 앱 저장공간(localStorage)에만 저장됩니다. 계정 생성, 광고, 분석 SDK, 원격 서버 또는 클라우드 동기화 기능은 없습니다.

권한 사용

알림 권한은 운동 사이 휴식 타이머의 남은 시간을 표시하는 데 사용됩니다. Foreground Service는 앱 밖에서도 사용자가 시작한 짧은 휴식 카운트다운을 유지하기 위해서만 동작합니다. Picture-in-Picture는 홈 화면이나 다른 앱 위에 해당 타이머를 표시할 때 사용됩니다.

백업과 삭제

사용자가 백업 메뉴에서 직접 내보낸 JSON 파일은 사용자가 선택한 위치에 저장되며 LogFit 서버로 전송되지 않습니다. 앱의 초기화 기능, Android의 앱 데이터 삭제 또는 앱 제거를 통해 기기 내 데이터를 삭제할 수 있습니다.

보안 및 아동

Android 앱은 OS 백업과 평문 네트워크 통신을 차단합니다. LogFit은 아동을 대상으로 설계되지 않았으며 사용자의 연령 정보를 수집하지 않습니다.

변경 및 문의

기능이나 데이터 처리 방식이 바뀌면 이 문서와 시행일을 갱신합니다. 문의는 LogFit GitHub 이슈를 이용해 주세요.
