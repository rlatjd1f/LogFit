# Foreground Service 및 권한 선언 초안

## 사용 유형

- Android 유형: `specialUse`
- subtype: `Workout timer for exercise rest intervals.`
- 사용자 시작점: 운동 기록 화면에서 세트의 `완료` 버튼을 누를 때
- 종료점: 카운트다운 완료, 앱의 건너뛰기/종료 동작, 또는 알림의 `타이머 종료` 버튼

## 사용자 혜택 설명

사용자가 세트를 완료한 뒤 다른 앱을 사용하거나 홈 화면으로 이동해도 정해진 휴식 시간을 놓치지 않도록 남은 시간을 지속적으로 표시합니다. 서비스는 운동 기록 자체를 추적하거나 위치·센서 데이터를 수집하지 않으며, 사용자가 시작한 최대 60분의 카운트다운 동안만 실행됩니다.

## 심사용 시연 절차

1. 앱 실행 후 홈 화면에서 `운동하기` 선택
2. 첫 운동의 `완료` 선택
3. 휴식 카운트다운과 알림 표시 확인
4. 홈 버튼을 눌러 PiP 타이머 확인
5. 알림 패널에서 남은 시간 및 `타이머 종료` 확인

Play Console의 Foreground service permissions 선언에는 위 설명과 실제 화면 녹화 영상을 제출합니다.
