# Google Play 심사자 안내

## 앱 개요

LogFit은 계정이나 네트워크 연결 없이 루틴을 만들고 세트별 중량·반복 수를 기록하는 운동 일지입니다. 테스트 계정은 필요하지 않습니다.

## 핵심 기능 확인

1. `운동하기`를 눌러 기본 루틴 진입
2. 세트의 중량·반복 수 조정 후 `완료` 선택
3. 자동 시작되는 휴식 타이머 확인
4. 홈 버튼을 눌러 PiP 타이머 확인
5. 알림 패널에서 휴식 남은 시간과 종료 동작 확인
6. `리포트`에서 운동 기록 요약 확인
7. `백업`에서 로컬 JSON 내보내기/가져오기 확인

## 권한 안내

- 알림: 운동 휴식 타이머를 앱 밖에서 표시
- Foreground Service specialUse: 사용자 시작 카운트다운 유지
- Picture-in-Picture: 다른 앱 사용 중 타이머 표시

건강 데이터 플랫폼, 위치, 카메라, 마이크, 연락처, 광고 식별자 권한은 사용하지 않습니다.
