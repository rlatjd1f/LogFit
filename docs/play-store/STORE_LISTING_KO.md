## 자세한 설명

LogFit은 웨이트 트레이닝 루틴과 세트 기록에 집중한 가벼운 오프라인 운동 일지입니다.

복잡한 가입이나 서버 연결 없이 바로 루틴을 시작하고, 운동별 중량과 반복 수를 기록할 수 있습니다. 세트를 완료하면 휴식 타이머가 자동으로 시작되어 다음 세트를 놓치지 않도록 도와줍니다.

주요 기능

• 분할 운동 루틴을 직접 만들고 순서대로 진행
• 운동별 중량, 반복 수, 세트 완료 기록
• 세트 완료와 동시에 휴식 카운트다운 시작
• 홈 화면이나 다른 앱 위에 PiP 휴식 타이머 표시
• 알림과 지원 기기의 상태 표시줄에서 남은 시간 확인
• 운동 기록과 점진적 과부하 리포트 확인
• JSON 파일을 이용한 오프라인 백업과 복구
• 라이트/다크 테마 지원

개인정보를 우선합니다

LogFit은 계정, 광고, 분석 SDK를 사용하지 않습니다. 루틴과 운동 기록은 기기에만 저장되며 개발자 서버로 전송되지 않습니다.

운동 중에는 기록에, 휴식 중에는 회복에 집중하세요.
