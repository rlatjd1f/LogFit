# 개인정보처리방침 배포

앱과 PWA에는 `/privacy.html`이 포함되어 있습니다. Play Console에 제출할 URL은 로그인 없이 HTTPS로 열리고 수정 불가능한 공개 페이지여야 합니다.

GitHub Pages를 이 저장소의 배포 대상으로 사용한다면 예상 URL은 다음과 같습니다.

`https://rlatjd1f.github.io/LogFit/privacy.html`

제출 전 체크:

- 실제 URL을 시크릿 브라우저와 모바일에서 열 수 있는지 확인
- 앱 안의 `개인정보처리방침` 링크가 동작하는지 확인
- Play Console의 개발자 연락처 이메일 입력
- 데이터 처리 방식 변경 시 정책과 Data safety 답변을 함께 갱신
