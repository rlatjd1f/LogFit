# Google Play Data safety 답변 초안

기준 버전: 0.5.0 (versionCode 5)  
작성일: 2026-07-21

## 데이터 수집 및 공유

- 앱에서 수집하거나 개발자/제3자 서버로 전송하는 사용자 데이터: 없음
- 제3자와 공유하는 사용자 데이터: 없음
- 전송 중 암호화: 서버 전송 자체가 없음
- 데이터 삭제 요청: 서버 보관 데이터가 없어 해당 없음. 사용자는 앱 초기화, Android 앱 데이터 삭제 또는 제거로 로컬 데이터를 삭제할 수 있음
- 계정 생성: 없음

## 기기 내부에서만 처리하는 데이터

- 루틴과 운동명
- 중량, 반복 수, 세트 완료 기록과 날짜
- 음향 등 앱 설정
- 진행 중 휴식 타이머 상태

위 데이터는 Android 앱 저장공간의 localStorage에만 저장됩니다. 사용자가 직접 JSON 백업을 내보내거나 가져올 수 있으며 백업 파일의 보관과 공유는 사용자 책임입니다.

## SDK 및 기능 확인

- 광고 SDK: 없음
- 분석/크래시 수집 SDK: 없음
- 로그인/계정 SDK: 없음
- 원격 서버 API: 없음
- Capacitor 및 AndroidX: 앱 런타임/UI 라이브러리이며 사용자 데이터를 개발자에게 전송하도록 구성하지 않음

Play Console 제출 직전에는 의존성과 네트워크 기능이 이 문서 작성 시점과 동일한지 다시 확인해야 합니다.
